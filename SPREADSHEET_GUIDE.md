# Googleスプレッドシート登録ガイド

## Empathy Timeline — 心象風景の推移 を表示するためのスプレッドシート構造

### 必要なシート

#### 1. `art` シート（作品データ）
作品の制作年が縦線として表示されます。

| 項目 | A | B | C | D | E | F | G | H |
|---|---|---|---|---|---|---|---|---|
| **列名** | year | title_jp | title_en | location | collection | image_url | description | insider_story |
| **説明** | 制作年（例: 1889） | 日本語タイトル（例: 星月夜） | 英語タイトル（例: The Starry Night） | 制作場所（例: サン＝レミ） | 所蔵美術館（例: ニューヨーク近代美術館） | 画像URL | 説明 | 深層背景 |
| **必須** | ✅ | ✅ |  |  |  |  |  |  |

**スプレッドシートにコピー&ペースト用（タブ区切り）：**

```
year	title_jp	title_en	location	collection	image_url	description	insider_story
1889	星月夜	The Starry Night	サン＝レミ	ニューヨーク近代美術館	https://example.com/image.jpg	ゴッホの代表作の一つ	精神療養院で制作された作品
```

**使い方：**
1. 上記のテキストをコピー
2. Googleスプレッドシートの`art`シートの1行目（A1セル）に貼り付け
3. タブ区切りが自動的に列に分割されます

#### 2. `Timeline` シート（感情年表データ）
画家の生涯における重要な出来事と感情の値を登録します。これがグラフの曲線として表示されます。

| 項目 | A | B | C | D | E | F |
|---|---|---|---|---|---|---|
| **列名** | artist_id | year | event | mood | description | category |
| **説明** | 画家ID（ArtistsシートのIDと一致） | 出来事が起きた年（例: 1888） | 出来事の説明（例: アルルへ移動） | 感情の値 | その時の状況や心理状態の説明 | カテゴリ |
| **必須** | ✅ | ✅ | ✅ | ✅ |  |  |
| **値の範囲** |  |  |  | -5 〜 +5 |  | personal / artistic / historical |

**mood（感情の値）の目安：**
- **+5**: 最高の幸福・歓喜
- **+3**: 大きな喜び・希望
- **+1**: ややポジティブ
- **0**: 中立・平穏
- **-1**: ややネガティブ
- **-3**: 大きな苦悩
- **-5**: 深い絶望・苦悩

**スプレッドシートにコピー&ペースト用（タブ区切り）：**

```
artist_id	year	event	mood	description	category
van-gogh	1880	画家の道を志す	2	伝道師としての挫折の後、絵画の世界へ。	personal
van-gogh	1888	アルルへ移動	5	「黄色い家」での共同生活。創作意欲の頂点。	personal
van-gogh	1888	耳切り事件	-5	ゴーギャンとの衝突、精神の崩壊が始まる。	personal
van-gogh	1889	サン＝レミの療養院	-2	「星月夜」を制作。苦悩の中の安らぎ。	artistic
```

**使い方：**
1. 上記のテキストをコピー
2. Googleスプレッドシートの`Timeline`シートの1行目（A1セル）に貼り付け
3. タブ区切りが自動的に列に分割されます

#### 3. `Artists` シート（画家情報）
画家の基本情報を登録します。periodから生年・没年を抽出して横軸の範囲を決定します。

| 項目 | A | B | C | D | E |
|---|---|---|---|---|---|
| **列名** | id | name | period | bio | avatar |
| **説明** | 画家ID（例: van-gogh） | 画家名（例: Vincent van Gogh） | 活動期間（例: Post-Impressionism (1853-1890)） | 略歴 | アバター画像URL |
| **必須** | ✅ | ✅ | ✅ |  |  |
| **形式** |  |  | (生年-没年)の形式 |  |  |

**periodの形式：**
- `(1853-1890)` のように、生年と没年を括弧内に記載してください
- これが横軸の範囲として使用されます

**スプレッドシートにコピー&ペースト用（タブ区切り）：**

```
id	name	period	bio	avatar
van-gogh	Vincent van Gogh	Post-Impressionism (1853-1890)	オランダの画家。情熱的で鮮やかな色彩と大胆な筆致で知られる。	https://example.com/avatar.jpg
```

**使い方：**
1. 上記のテキストをコピー
2. Googleスプレッドシートの`Artists`シートの1行目（A1セル）に貼り付け
3. タブ区切りが自動的に列に分割されます

### スプレッドシートの設定

1. **シート名を正確に設定**
   - `art`（小文字）
   - `Timeline`（大文字T）
   - `Artists`（大文字A）

2. **1行目はヘッダー行として使用**
   - 2行目からデータを入力してください

3. **公開設定**
   - 「リンクを知っている全員が閲覧可能」に設定してください

### グラフの表示内容

- **横軸**: `Artists`シートの`period`から抽出した生年〜没年、または`Timeline`シートの最初と最後の年
- **縦軸**: 感情の強度（-5 〜 +5）
- **曲線**: `Timeline`シートのデータポイントを滑らかに接続
- **縦線**: `art`シートの各作品の`year`に縦線を表示
  - 現在表示中の作品の年は強調表示（太いアンバー色の線）
  - その他の作品の年は薄いグレーの線

### データ入力のコツ

1. **Timelineシート**
   - 画家の生涯における重要な転機を5〜10個選んで登録
   - 作品が制作された年も含めると良い
   - moodの値は、その出来事が画家に与えた影響を反映

2. **artシート**
   - 作品の制作年を正確に入力（縦線の位置が決まります）

3. **Artistsシート**
   - periodは `(生年-没年)` の形式で入力（例: `(1853-1890)`）

### 注意事項

- データが空の場合は、AI（Gemini API）で自動生成されます
- スプレッドシートのデータが優先されます
- シート名は大文字小文字を区別します
